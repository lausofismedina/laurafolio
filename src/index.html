Robofriends

Creé una applicación web con React simple de simulación de base de datos de robots. Cada tarjeta de robot se genera con un ID aleatorio y muestra el nombre y el correo electrónico del robot. La aplicación incluye un componente de tarjeta que muestra la imagen del robot utilizando el servicio 'robohash.org' con el ID generado. Además, la aplicación cuenta con una función de búsqueda que permite filtrar las tarjetas de robots según el nombre o el correo electrónico.

Neural Networks for Image Recognition

Implementé y evalué una red neuronal con los datos de Colab CIFAR-10 usando capas convolucionales, capas de dropout para regularización, capas densas, y capas de salida con activación softmax. Comparé el rendimiento de esta arquitectura con otra, evalue su eficacia en la clasificación de imágenes. Cargué tres imágenes diferentes a la de la guitarra y las clasifiqué. Luego, visualicé la salida de una de las imágenes para cada uno de los 64 filtros encontrados por la red en la primera capa de convolución. Apliqué Maxpooling y corroboré manualmente las salidas de convolución y MaxPooling para uno de los filtros utilizando numpy y mi propio código, sin usar funciones de convolución predefinidas.
